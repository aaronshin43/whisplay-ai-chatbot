Category: Impaled Object

STEPS:
1. Call emergency services (911/999/112) immediately.
2. Do NOT remove the object under any circumstances — it may be preventing fatal hemorrhage.
3. Stabilize the object so it cannot move: use bulky padding (folded cloths, bandages) built up around both sides of the object.
4. Secure the padding gently with bandages to hold the object still — do not press down on it.
5. Control bleeding around the object by pressing on the surrounding tissue, not directly on the object.
6. Keep the person still and as calm as possible.
7. If the impaled object is very long, shorten it only if it is absolutely necessary to move the patient and cannot be done otherwise — never pull it out.
8. Monitor for shock while waiting for emergency services.

NEVER DO:
- Do NOT pull out the impaled object regardless of its size.
- Do NOT press directly on the object.
- Do NOT transport the person without stabilizing the object first.
- Do NOT attempt to shorten the object unless absolutely necessary for rescue.
